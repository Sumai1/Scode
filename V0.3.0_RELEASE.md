# 🎉 Scode v0.3.0 - 重大升级！

## 新增功能

### ✅ 规划模式
- `/plan` 命令创建任务计划
- AI 自动分解任务为步骤
- 显示清晰的执行计划
- 支持退出规划模式

**使用方法：**
```bash
You: /plan
Enter task to plan: 创建一个 REST API 项目
📋 Plan:
1. 创建项目结构
2. 添加依赖
3. 实现路由
4. 添加中间件
5. 编写测试
```

### ✅ 配置文件支持
- 配置文件位置：`~/.scode/config.toml`
- 支持 API、Agent、UI 配置
- 命令行参数优先级更高

**配置示例：**
```toml
[api]
api_key = "your-key"
base_url = "http://your-endpoint"
model = "claude-sonnet-4-5-20250929"

[agent]
max_iterations = 20
auto_approve_read_only = false

[ui]
colors = true
show_tokens = true
show_progress = true
```

**初始化配置：**
```bash
./target/release/scode --init-config
```

### ✅ 新增 3 个 Git 工具
- `git_branch` - 分支管理（列出/创建/切换）
- `git_pull` - 拉取远程更改
- `git_push` - 推送到远程仓库

### ✅ 新增命令
- `/plan` - 进入规划模式
- `/exit-plan` - 退出规划模式
- `/config` - 显示当前配置

## 使用示例

### 规划模式

```bash
./target/release/scode

You: /plan
Enter task to plan: 重构项目代码

📋 Plan:
1. 分析现有代码结构
2. 识别需要重构的部分
3. 创建新的模块结构
4. 逐步迁移代码
5. 运行测试确保功能正常

You: 开始执行第一步
🤖 Scode: [开始分析代码...]
```

### 配置文件

```bash
# 初始化配置
./target/release/scode --init-config

# 编辑配置
vim ~/.scode/config.toml

# 查看配置
./target/release/scode
You: /config
```

### Git 工作流

```bash
./run.sh "创建新分支 feature/new-api，切换到该分支，然后推送到远程"
```

AI 会自动：
1. `git_branch` create feature/new-api
2. `git_branch` switch feature/new-api
3. `git_push` origin feature/new-api
