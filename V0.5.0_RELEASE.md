# 🎉 Scode v0.5.0 - 最终版本！

## 新增功能

### ✅ Web Search 工具
- `web_search` - 使用 DuckDuckGo 搜索网页
- 返回搜索结果标题和摘要
- 支持自定义结果数量

### ✅ HTTP 请求工具
- `http_request` - 发送 HTTP 请求
- 支持 GET, POST, PUT, DELETE
- 自定义 headers 和 body
- 完整的 API 调用支持

### ✅ 文件系统工具（5个新增）
- `file_list` - 列出目录内容（支持递归）
- `file_move` - 移动/重命名文件
- `file_copy` - 复制文件
- `file_delete` - 删除文件/目录
- `file_info` - 获取文件详细信息

## 使用示例

### Web Search

```bash
./run.sh "搜索 Rust 异步编程最佳实践"
```

AI 会使用 `web_search` 工具搜索并总结结果。

### HTTP API 调用

```bash
./run.sh "调用 https://api.github.com/users/octocat 获取用户信息"
```

AI 会使用 `http_request` 工具发送请求。

### 文件管理

```bash
./run.sh "列出 src/ 目录的所有文件，然后复制 main.rs 到 main.rs.bak"
```

AI 会使用 `file_list` 和 `file_copy` 工具。
