### 🚀 快速开始

**1. 初始化配置：**
```bash
cd /root/.openclaw/workspace/scode
./target/release/scode --init-config
```

**2. 编辑配置：**
```bash
vim ~/.scode/config.toml
```

**3. 使用规划模式：**
```bash
./target/release/scode

You: /plan
Enter task to plan: 创建一个 web 服务器
[AI 生成计划]

You: 开始执行
[AI 按计划执行]
```

**4. 使用新的 Git 工具：**
```bash
./run.sh "创建新分支 feature/api，切换过去，然后推送"
```
