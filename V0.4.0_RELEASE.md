# 🎉 Scode v0.4.0 - 重大升级！

## 新增功能

### ✅ 彩色 UI
- 彩色输出，更好的视觉体验
- 可通过配置或 `--no-color` 禁用
- 不同类型的消息使用不同颜色
- 进度条和 spinner 支持

### ✅ 子 Agent 系统
- `/spawn` 命令创建子 Agent
- 子 Agent 独立执行任务
- 支持并行执行多个子 Agent
- 任务状态跟踪

### ✅ 进度显示
- 工具执行进度条
- Spinner 动画
- 实时状态更新

### ✅ 新增命令
- `/spawn` - 创建子 Agent
- `/agents` - 列出所有子 Agent

## 工具总览

**15 个工具**（与 v0.3.0 相同）

## 使用示例

### 彩色 UI

```bash
# 默认启用彩色
./target/release/scode

# 禁用彩色
./target/release/scode --no-color
```

### 子 Agent

```bash
./target/release/scode

You: /spawn
Enter sub-agent task: 分析 src/ 目录的代码质量
✓ Spawned sub-agent: abc-123-def
ℹ Running sub-agent...
✓ Sub-agent completed
Result: [分析结果]

You: /agents
ℹ Sub-agents (1):
  abc-123-def - Completed: 分析 src/ 目录的代码质量
```

### 并行任务

```rust
// 在代码中使用
let results = agent.run_subagents_parallel().await;
```
